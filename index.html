Casos de Estudio Detallados

Landing Page Interactiva con Carrusel 3D

Fecha / Duración del Proyecto: Mayo - Junio 2024

El Reto / Problema:

Crear una demo de landing page visualmente atractiva y moderna, que fuera más allá de un carrusel estándar, implementando un componente interactivo y dinámico (carrusel circular 3D con efectos de hover) para mostrar habilidades avanzadas de frontend y la capacidad de crear experiencias de usuario únicas.

Mi Rol y Responsabilidades:

Desarrolladora Frontend Principal. Encargada de la implementación del carrusel circular 3D personalizado con CSS y JavaScript puro, así como del diseño y maquetación de las secciones de servicios, formulario de contacto y footer utilizando Bootstrap. Aseguré la responsividad en diferentes dispositivos.

La Solución y Proceso:

Se optó por una arquitectura de página única. El carrusel 3D se desarrolló desde cero utilizando CSS para las transformaciones 3D y la animación de rotación. Se implementó JavaScript para el posicionamiento dinámico de los elementos y la interactividad al pasar el ratón, incluyendo el efecto de zoom y la visibilidad de los captions. Se definieron media queries para asegurar un diseño responsivo y adaptativo.

  • Tecnologías Usadas: HTML5 CSS3 JavaScript (Vanilla JS) Bootstrap 5
  • Metodología: Desarrollo Iterativo / Prototipado Rápido
  • Herramientas: VS Code, Navegador (para depuración), GitHub (para control de versiones)

Resultados e Impacto:

Se creó una demo de Landing Page altamente interactiva y visualmente impactante, destacando la capacidad de desarrollar componentes frontend complejos y personalizados. El carrusel circular 3D sirve como un diferenciador clave, atrayendo la atención y mostrando una comprensión profunda de CSS y JavaScript. La demo es completamente responsiva, asegurando una experiencia de usuario consistente en dispositivos de escritorio y móviles.
